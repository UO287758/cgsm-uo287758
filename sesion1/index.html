Memoria

Descripción de la práctica

Descripción breve del desarrollo de la práctica respondiendo a las cuestiones que se plantean en el guión.

Ejercicio 1

En este ejercicio se crea un archivo css y html basico para comprobar que la instalacion de npm funciono correctamente. Ademas sirve de base.

Ejercicio 2

En este ejercicio se explica qué herramientas usar y cómo organizar el proyecto para poder ejecutar y probar el código WebGL correctamente.

Ejercicio 3

En este ejercicio se aprende a crear un canvas en HTML y a obtener el contexto WebGL desde JavaScript. También se introducen los shaders.

Ejercicio 4

En este ejercicio se explica cómo crear y dibujar objetos mediante vértices y buffers y se usa para crear unas figuras en 3D.

Ejercicio 5

EN este ejercicio se introducen las transformaciones matemáticas que permiten mover los objetos. Se utilizan matrices para cambiar la posición, orientación y tamaño de los objetos dentro de la escena.

Ejercicio 6

En este ejercicio se añade iluminación para que los objetos se vean más realistas. Se explica cómo calcular la luz sobre las superficies y cómo los shaders pueden modificar el color de los objetos según la iluminación.

Dificultades encontradas

En lo que mas dificultades tuve fue en poder entender como se posicionan las figuras, como se introducen las luces para poder dar textura a los componentes, y todo lo que tenga que ver con persperctivas en general. Las luces fueron especialmente complicadas porque ademas de tener que entender donde se posiciona la luz, hay que saber que luz usar para dar el efecto que se quiere.
